\subsection{Detailed Description}
Abstract handler class for implementing actuators. 

\hyperlink{classrlib_1_1ActuatorHandler}{Actuator\-Handler} objects are used by the \hyperlink{classrlib_1_1Actuator}{Actuator} class for commanding actuator hardware.
