\subsection{Detailed Description}
Abstract class for creating \hyperlink{classrlib_1_1Sensor}{Sensor} objects. 

This class defines the abstract class that all \hyperlink{classrlib_1_1Sensor}{Sensor} derivatives must follow. It is intended for handling representation details of any sensor. It must have a \hyperlink{classrlib_1_1SensorHandler}{Sensor\-Handler} in order to interact with a physical sensor.
