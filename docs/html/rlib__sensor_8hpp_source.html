1 // Robot Library v0.1
2 // This is a library for contructing robot objects for
3 // doing basic tasks such as sensing, navigation, and planning.
4 //
5 // This library file is for constructing the sensor classes.

8 // specific sensors with their own sensing needs such as Lidars, Sonars,
9 // Cameras, etc.
