1 // Robot Library v0.1
2 // This is a library for contructing robot objects for
3 // doing basic tasks such as sensing, navigation, and planning.
4 //
5 // This library file is for constructing the actuator classes.
6 // One of the main ideas in this library is that actuators can be
7 // constructed into chains of actuators using the parent child
8 // structure. If the actuator needs more than one child then
9 // the robot class in the rlib_robot.cpp library can be used
10 // to allow multiple children and to add additional structures
11 // such and sensors and planners.
12 //
13 // Actuator is an abstract class to allow for the creation of
14 // specific actuators with their own movement needs such as mobile
15 // bases, servos, and manipulators.
16 //
17 // Author: Griswald Brooks
18 // Email: griswald.brooks@gmail.com
19 //
