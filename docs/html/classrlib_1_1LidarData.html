Detailed Description

LidarData class for getting data from a Lidar Sensor.

This class is for encapsulating the data produced by a Lidar Sensor and making it easy to access the data in different ways.

Constructor for LidarData objects.

*** Lidar Data Class ***///

The constructor for the needs a std::vector of ranges and angles. Each vector must be the same length and the indices must correspond with each others data. The vectors do not need to be sorted by angle, and in general are not.

Parameters
[in]rangesThis is the IP address of the computer that the Lidar is connected to.
[in]anglesThis is the port number of the Lidar on the computer.
