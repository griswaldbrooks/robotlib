Detailed Description

Hokuyo Lidar Handler class using IP connected Lidar.

This is a handler class derived from the rlib::SensorHandler class. It is designed for use with the Hokuyo Lidar URG-04LX-UG01. The Lidar is started using the Python lidar_server_tester.py written by Abhimanyu Ghosh. This client handler is a wrapper class for the client library hokuyo_client.cpp written by Brian Cairl.
